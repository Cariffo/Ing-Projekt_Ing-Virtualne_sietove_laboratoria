-Set static IP address (faster boot times)
    vim /etc/network/interfaces


    auto eth0
    iface eth0 inet static
      address X.X.X.X
      netmask X.X.X.X
      gateway X.X.X.X


    :wq

-Set DNS nameservers; the "dns-nameservers" option in 
"/etc/network/interfaces" didn't work.

    sudo vim /etc/resolvconf/resolv.conf.d/base


    nameserver 8.8.8.8
    nameserver 8.8.4.4


    :wq

    sudo resolvconf -u
    cat /etc/resolv.conf
    ping google.com

    sudo apt-get install aptitude bash-completion
    reboot
